## Rules for the daily agent

1. **Never duplicate a slug** that already exists in `content/blog/`.
2. **Pick from this backlog** when possible. If all topics here are exhausted, brainstorm new ones in the same niches.
3. **Mark used topics** by listing them in the commit message — don't edit this file unless needed.
4. **Prioritize ★★★ topics** over ★★ and ★ where free choice exists.
5. **Mix clusters daily** — never write 5 articles from the same cluster on the same day. Aim: 2 memory + 2 quran + 1 kids, or similar mix.
